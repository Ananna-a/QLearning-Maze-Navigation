# Python强化学习迷宫程序设计 作业任务需求

## 核心任务目标

1. **环境建模**：理解状态空间（State）、动作空间（Action）与奖励机制（Reward）的构建。

2. **算法实现**：手动实现 Q\-Learning 算法，掌握 Q\-table 的更新逻辑。

3. **可视化展示**：利用 Tkinter 或 Pygame 实时演示智能体从 \&\#34;随机探索\&\#34; 到 \&\#34;最优路径\&\#34; 的演化。

## 作业内容：软件架构设计

本设计建议采用模块化结构，将逻辑与显示分离，分为以下三个核心模块：

### 1\. 迷宫环境模块 \(maze\\\[\_env\.py\]\(\_env\.py\)\)

使用 Gym 风格的接口，包含以下核心要素：

- **状态 \(State\)**：智能体在坐标系中的位置 \(x, y\)。

- **动作 \(Action\)**：上下左右，对应编码 \[0, 1, 2, 3\]。

- **奖励 \(Reward\)** 规则：

    - 到达终点：\+1

    - 撞墙 / 掉入陷阱：\-1

    - 每走一步：\-0\.01（引导智能体寻找最短路径）

### 2\. 强化学习大脑模块 \(rl\\\[\_brain\.py\]\(\_brain\.py\)\)

核心是维护一张 Q\-table，其核心更新公式为：
$Q(s, a) \leftarrow Q(s, a) + \alpha \ [r + \gamma \ \max_{a'} Q(s', a') - Q(s, a)]$

### 3\. 主循环控制 \(\[main\.py\]\(main\.py\)\)

- 负责训练轮次（Episodes）的迭代

- 控制训练速度与日志输出
